Integrated Gradients

Integrated gradients is a popular method for feature attribution: computing which features were most important towards the output prediction. The \(i\)th feature has importance \(\phi_i^{IG}(f, x, x')\), where: $$\phi_i^{IG}(f, x, x') = (x_i - x'_i) \times \int_{\alpha = 0}^ 1 \frac{\delta f(x' + \alpha (x - x'))}{\delta x_i} d \alpha$$ We calculate the feature importance for every pixel in the input image \(x\) and plot the result below in a saliency map. Lighter pixels indicate more influential ones. As a baseline \(x'\), we use the constant black image.


Some of the pixel attributions may seem unintuitive. But why? To understand why integrated gradients highlights the pixels it does, we will need to understand where the above equation comes from, and what it is computing.
