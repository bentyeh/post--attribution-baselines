Gradient Saturation

Integrated gradients is defined as: $$\phi_i^{IG}(f, x, x') = (x_i - x'_i) \times \int_{\alpha = 0}^ 1 \frac{\delta f(x' + \alpha (x - x'))}{\delta x_i} d \alpha$$ What does this formula do? Derivatives represent rate of change. Pixels with the largest magnitude derivatives are those that, when perturbed, would most change the output prediction. But the derivatives with respect to certain features may be small even if those features are very important towards a prediction, because neural networks are prone to saturation. That is, the output function may flatten after input features pass a critical value.
