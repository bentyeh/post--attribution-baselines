SLIC as a Whistle

When we run integrated gradients, we are accumulating gradients between some baseline \(x'\) and our current input \(x\). This \(x'\) is meant to represent "missingness". How can we represent missingness using fixed feature values? Each feature value has meaning - in the case of images, each feature value corresponds to a color.


Because the baseline we choose is fixed, integrated gradients becomes blind to the baseline. That is, if we use the black image as a baseline, integrated gradients will not highlight black pixels as important! This can be a major problem if the object of interest contains the color that you used as a baseline.
